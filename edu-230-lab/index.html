EDU-230 · LABS 1–8

Cyberthreat Protection

8 hands-on labs covering SSL inspection, DNS security, URL filtering, browser isolation, ZPA traffic inspection, sandbox file analysis, real-time alerting, and deception-based active defense.

These labs walk you through building a layered cyberthreat protection strategy using Zscaler Internet Access (ZIA), Zscaler Private Access (ZPA), and Zscaler Deception. Start from Lab 1 and work sequentially — each lab builds on configurations from the previous one. Labs 7 and 8 include SDC (Solutions Demo Center) read-only tasks that use pre-populated data.
